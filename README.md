Get code
```
[git clone https://github.com/wouter-de-boer/opentrivia-java-spring-boot.git](https://github.com/wouter-de-boer/opentrivia-react.git)
```

Build & Run
```
cd opentrivia-react
```
```
npm install
```
```
npm run build
```
```
serve -s build
```
Or
```
npm run start
```
